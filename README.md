# assert

Test Assertion Helpers

`assert` is a lightweight Go library providing a simple and expressive set of test assertion helpers. It wraps Go's standard testing.T to offer clean, readable assertion functions, such as equality checks, nil/not-nil validation, boolean expectations, and error handling. This reduces boilerplate in unit tests and producing clear, descriptive failure messages when assertions do not pass.

This library is a direct implemenation of the the [article](https://www.alexedwards.net/blog/the-9-go-test-assertions-i-use)
